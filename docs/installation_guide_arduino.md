# How to transform your Arduino into an Speech Emotion Recognition Device with Edge Impulse

Welcome to this tutorial of how to create your own Speech Emotion Recognition (SER) device, at the end of this tutorial you will know how to implement your own tiny machine learning model for speech emotion recognition with **Edge Impulse** and implement it in real life on your Arduino Nano 33 BLE sense rev2. 

This tutorial is part of the **MADI PROJECT** submitted to the Edge Impulse competition of Hackearth.
